# Отчёт по проекту "Kolobok: escape from Grandma" - игры раннера

Проект выполнил студент 4 курса Окунев Данила (БПМИ225)

Игра представляет собой 3D-раннер с видом от третьего лица, в котором игрок (жёлтая сфера) автоматически движется вперёд по бесконечной платформе.
Игрок должен уклоняться от препятствий (синие и голубые пирамиды-шипы разных размеров и настраеваемых форм), перемещаясь по трём линиям (как в Subway Surfers), 
и собирать бонусы (красные и золотые капсулы), увеличивая свой *score*.

**MainMenu** - Сцена главного меню, где можно начать новый забег, увидеть свой лучший скор (*record*) или выйти из игры.      
**GameScene** - основная сцена в проекте, содержащая все игровые объекты.      
**GameManager.cs** - главный скрипт, отвечающий за управление состоянием игры (перезапуск, завершение).

## Описание реализации

### 1. Система игрока
- **PlayerController** - обрабатывает ввод с клавиатуры (A/D для смены линии, Space для прыжка)    
- **CharacterController** (как часть предыдущего скрипта) - используется для физики перемещения и коллизий   
- **PlayerHealth** - управление здоровьем, получением урона и лечением

### 2. Система препятствий
- **ObstacleSpawner** - периодически генерирует препятствия впереди игрока
- **Obstacle** - два типа препятствий (разные prefab'ы с разным уроном)
- Препятствия появляются в одной из трёх линий случайным образом
- Автоматическое удаление пройденных препятствий

### 3. Система бонусов
- **BonusSpawner** - генерация бонусных предметов
- **Boneses** - два типа бонусов:
  - **Красные капсулы** (BonusType.Heal) - восстанавливают здоровье
  - **Золотые капсулы** (BonusType.Invincibility) - дают временную неуязвимость

### 4. Генерация мира
- **GroundSpawner** - создаёт бесконечную дорожку из тайлов
- **FollowCamera** - камера следует за игроком с плавным движением

### 5. Пользовательский интерфейс
- **UIManager** - отображение здоровья, скорости, скора и статуса неуязвимости
- **MainMenuManager** - окно главного меню с названием игры, автором, рекордом и кнопками начала забега/выхода из игры.

### 6. Система очков (score'а)
- **ScoreManager** - управление очками игрока (их отображение и сохранение), которые зарабатываются со временем и при подборе бонусов.

### 6. Анимации
- **PlayerAnimationController** - управление анимациями игрока (при получении урона, подборе бонусов: временных и или немедленных).

## Выполненные дополнительные функции

### ✓ Бонусы
Реализованы два типа бонусов:
- **Лечение** - восстанавливает здоровье игрока
- **Неуязвимость** - делает игрока временно неуязвимым к урону

### ✓ Ускорение игры со временем
- **GameSpeedManager** постепенно увеличивает скорость движения игрока
- Скорость растёт от `startSpeed` до `maxSpeed` с шагом `stepAmount` каждые `stepInterval` секунд
- Текущая скорость отображается в UI

### ✓ Анимации
Реализованы три вида анимации:
- **При получении урона**: игрок начинает быстро мигать красным
- **При лечение**: игрок начинает быстро мигать зелёным
- **При неуязвимости**: игрок становится серого (как сталь) цвета, иногда мигая бирюзовым. Эффект заканчивается вместе с неуязвимостью

## Особенности реализации

1. **Предотвращение наложения объектов** - проверка коллизий перед спавном препятствий и бонусов
2. **Настраиваемость** - все главные параметры доступны для настройки в Inspector (за исключением самых базовых)
3. **Корректное удаление объектов** - препятствия и бонусы уничтожаются после прохождения

## Заимствования

Код написан с большего самостоятельно, лишь с небольшой помощью https://chatgpt.com/, который использовался на начальных этапах разработки 
(для помощи в создани скриптов перемещения и следования камеры), однако весь код разбирался, тестировался, и редактировался уже самим разработчиком (т.е. мной).
Основной источник материалов и знаний, который использовался при создании данного проекта, это заняти курса "Основы разработки компьютерных игр", в рамках которого этот проект и был написан.
Все ассеты (модели, материалы) являются стандартными примитивами Unity (сфера, капсулы, кубы) с применением вручную созданных "материалов" (различных цветов).
